# 🧚‍♀️ Fairy Nails by Valu 💅✨

## 📌 Descripción

Fairy Nails by Valu es una página web creada para mostrar productos y servicios relacionados con el mundo de las uñas. 
El diseño está inspirado en un estilo mágico y femenino, utilizando colores suaves y elementos de fantasía.

## 🚀 Funcionalidades

- 🛍️ Visualización de productos.
- 🛒 Carrito de compras interactivo.
- ✏️ Edición y eliminación de productos del carrito.
- 💾 Persistencia del carrito mediante LocalStorage.
- 📩 Formulario de contacto funcional con Formspree.
- 📱 Diseño adaptable para distintos dispositivos.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
-   ## 🎨 Diseño

La página utiliza una estética inspirada en hadas, con tonos rosas y detalles delicados para representar la identidad de la marca.
